# Password Complexity Validator

## Description
Password Complexity Validator is a simple Java project based on Object-Oriented Programming concepts. The main purpose of this project is to check whether a user’s password is strong or not based on basic security rules.

The system takes a username and password as input and validates the password using conditions such as minimum length (at least 8 characters), presence of at least one numeric digit, and at least one uppercase letter. If all conditions are satisfied, the password is considered valid; otherwise, it is marked as invalid.

This project is divided into multiple classes to maintain proper structure and separation of concerns. The UserAccount class is used to store user data, while the SecurityValidator class handles all validation logic.

This project helped me understand important Java concepts like classes, objects, encapsulation, and string handling.


## Features
- Minimum 8 characters
- At least 1 digit
- At least 1 uppercase letter

## Technologies Used
- Java
- OOP Concepts
